نصب زنجیر روی موبایل

برای استفاده از زنجیر روی گوشی، میتونید از اپلیکیشن Element استفاده کنید.

1

دانلود Element

اپلیکیشن Element رو از کافه‌بازار، مایکت، یا Google Play نصب کنید.

2

انتخاب سرور سفارشی

در صفحه ورود، "Other" یا "سرور دیگر" رو انتخاب کنید.

3

وارد کردن آدرس سرور

آدرس سرور رو وارد کنید و وارد حساب کاربری خود بشید.
